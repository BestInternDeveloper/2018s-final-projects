1. 对TVM的架构、思想进行学习
2. 从中总结程序设计思想、性能优化思想、tradeoff等
3. 对TVM的使用进行测试 -- [项目链接](https://github.com/Alexyuanmu/fopl-2018-TVM-learning.git)
